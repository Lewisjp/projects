JavaScript Munchkins: Function Expressions Declared functions, like the Plutonium Dragon from the last post, are built in memory right as we load the program. Function expresessions are assigned to a varaible so they'll only be loaded when its reached inside the program. // Syntax of a Function Expression var variable = function functionNameIsOptional (parameters) { code to be executed }; // example of calling a Function Expression variable(parameters); You'll notice above that the variable name includes parentheses, and we've included parameters to be used by the function it contains.
